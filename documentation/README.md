# Simpl-Local documentation

Repo-wide reference documents for the Simpl-Local evaluation workspace. Per-stack documentation
lives under each subproject's own `docs/` folder; this folder holds cross-cutting material that is
not tied to a single stack.

| Document | What it covers |
|---|---|
| [`simpl-dataspace-landscape.md`](./simpl-dataspace-landscape.md) | Non-technical orientation guide to the European data-space landscape: how Simpl, Gaia-X, DSSC, IDSA, EDC/Eclipse DSWG, Tractus-X, Catena-X, Prometheus-X, X-Road, and FIWARE relate, with a layer map and a who-does-what table. |
| [`simpl-dssc-mapping.md`](./simpl-dssc-mapping.md) | Full mapping of Simpl-Open onto the DSSC Data Spaces Blueprint V3.0 (key concepts and building blocks), with diagrams, coverage ratings, divergences, and standards alignment. |
| [`simpl-tractusx-comparison.md`](./simpl-tractusx-comparison.md) | Technical comparison of Simpl-Open with Eclipse Tractus-X / Catena-X: shared EDC and DSP foundations, identity and semantics divergences, component mapping, where the programmes already interact, and what Tractus-X's local-evaluation tooling means for simpl-local. |
